###資料標準化
當資料的量級差距過大，可能會造成神經網路較重視數值大的資料，而忽略數值小的資料。
使用標準分數，給定一個平均數後，計算距離平均數有幾個標準差，大於平均數為正數，小於為負，讓數值盡量在0的附近
